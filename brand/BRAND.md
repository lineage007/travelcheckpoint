# TravelCheckpoint — Brand Reference

## Logo
**Wordmark:** "TravelCheckpoint" — light "Travel" + bold "Checkpoint"
- **Font:** Outfit (Google Fonts)
- **Weight:** Travel = 300 (light, 70% opacity) | Checkpoint = 700 (bold, 100% opacity)
- **Kerning:** -0.02em

### Logo Files
| File | Use |
|------|-----|
| `brand/logo-white-on-dark.png` | For dark backgrounds (primary — website default) |
| `brand/logo-dark-on-light.png` | For light backgrounds (documents, print) |
| `public/logo-white.png` | Deployed on website (header) |
| `public/logo-dark.png` | Deployed on website (light contexts) |

### Logo Rules
- Do NOT change the font (Outfit)
- Do NOT change the light/bold weight split
- Do NOT add icons or marks to the logo
- Minimum clear space: 1x the height of the "C" in Checkpoint on all sides

## Typography
| Use | Font | Weight |
|-----|------|--------|
| Display / Headlines | Outfit | 600-700 |
| Body | Outfit | 400-500 |
| Mono / Prices / Codes | JetBrains Mono | 400-600 |

## Design Language
- Dark-first — all backgrounds are near-black
- Glassmorphism — frosted glass cards with subtle borders
- Mesh gradients — animated purple/cyan/pink background orbs
- Pill-shaped badges and buttons for filters
- Staggered fade-in animations on page load
- Destination cards use vivid gradient overlays
